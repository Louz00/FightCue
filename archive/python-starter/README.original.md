# FightCue

Standalone starter project for `FightCue`.

## Included

- Independent Python package layout under `src/`
- Minimal CLI entry point
- Basic test scaffold
- Separate project metadata in `pyproject.toml`

## Quick start

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -e .[dev]
python -m fightcue.app
pytest
```
